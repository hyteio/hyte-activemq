# HYTE ActiveMQ

HYTE enhancements to Apache ActiveMQ for client and server features.

## Background

HYTE has been enabling high-performance ActiveMQ for organizations for over a decade. These enhancements are made available to the community to prove out the functionality, usefulness and performance of these extensions.

Many of these will be contributed back to the Apache ActiveMQ project as core modules.
